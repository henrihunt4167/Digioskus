# Digioskus
1. GitHub Education konto

Mine: https://education.github.com/students

Logi sisse oma GitHubi kontoga

Kinnita, et oled üliõpilane (kooli e-post vms)

2. Loo uus hoidla (repository)

GitHubis vali New repository

Täida väljad:

Repository name: Harjutamine

Description:

Tegemist on versioonihalduse harjutamisega GitHubis.

Visibility: Public ✅

Märgi Add a README file ✅

Vajuta Create repository

3. README.md sisu (soovituslik)

Ava README.md ja kirjuta näiteks:

See hoidla on loodud GitHubi versioonihalduse harjutamiseks.
Siin harjutatakse harusid (branches), failide muutmist ja ühendamist (merge).


Salvesta (Commit changes → main).

4. Loo fail põhiharus (main branch)

Veendu, et oled main harus

Vali Add file → Create new file

Faili nimi: harjutamine.txt

Lisa AINULT GitHubi tutvustus:

GitHub on veebipõhine platvorm, mis võimaldab tarkvaraarendajatel ja teistel kasutajatel hallata ning jagada lähtekoodi. 
GitHubi kasutavad programmeerijad, meeskonnad, ettevõtted ja avatud lähtekoodiga projektide arendajad üle maailma.


Commit:

Commit message: Lisa GitHubi tutvustus

Branch: main

5. Loo kõrvalharu (branch)

Branch menüüst vali New branch

Nimi: minu-motted

Loo haru main harust

6. Muuda faili kõrvalharus

Veendu, et oled harus minu-motted

Ava harjutamine.txt

Lisa alla järgmised osad:

a) Sinu mõtted GitHubi kasutamisest (1–2 lauset)

(Võid kasutada seda teksti või kohanda)

Edaspidi saan GitHubi kasutada koolitööde ja isiklike programmeerimisprojektide hoidmiseks ning versioonide haldamiseks.
Samuti võimaldab GitHub meeskonnatööd ja koostööd teiste arendajatega.

b) GitHubi ajaloo lõik
GitHub Inc asutati 2008. aastal Tom Preston-Werneri, Chris Wanstarthi, PJ Hyetti ja Scott Chaconi poolt.
GitHubi teenus muutus avalikult kättesaadavaks 2008. aastal ning 2018. aastal omandas ettevõtte Microsoft.
